# Progressive Disclosure Structure Plan

This document outlines how content should be revealed progressively across pages to avoid reader repetition while maintaining engagement.

## Content Depth Levels

### Level 1: Homepage (Entry Point - Teaser)
**Purpose:** Hook, orient, invite deeper exploration  
**Content Strategy:**
- Brief, compelling introduction only
- ONE unique quote (not repeated elsewhere)
- High-level concepts mentioned but not explained
- Teaser excerpt (different from other pages)
- Clear CTAs to go deeper

**What to REMOVE from homepage:**
- Full "Paradigm" section (move to About-the-book)
- Detailed explanations
- Multiple quotes
- Long manuscript excerpts

**What to KEEP on homepage:**
- Hero with subtitle
- Brief "What Book One Contains" (one sentence)
- ONE short manuscript glimpse (unique excerpt)
- "Who This Is For" (brief list)
- "About This Work" (condensed version)
- CTA section

---

### Level 2: Books Page (More Detail)
**Purpose:** Show what's inside, build interest  
**Content Strategy:**
- Deeper exploration of book content
- Multiple quotes (different from homepage)
- Specific chapter/interlude previews
- Testimonials from readers
- More detailed excerpts

**What to REMOVE from books page:**
- "Paradigm" explanation (already on homepage/About-the-book)
- Full philosophical explanations
- Quotes that appear on homepage

**What to KEEP/ADD on books page:**
- Book cover and description
- "Report from the Field" (testimonials)
- Multiple manuscript excerpts (different ones)
- Chapter previews
- Book Two/Three teasers
- Different quotes than homepage

---

### Level 3: About-the-Book Page (Full Depth)
**Purpose:** Complete understanding, deepest content  
**Content Strategy:**
- Full philosophical explanations
- Complete quotes with full context
- Deepest level of detail
- Full manuscript excerpts
- Complete "Paradigm" section

**What to REMOVE from About-the-book page:**
- Brief teasers (those belong on homepage)
- Simple descriptions (those belong on books page)
- Quotes already shown on homepage/books

**What to KEEP/ADD on About-the-book page:**
- Full "Paradigm" section (moved from homepage)
- Complete "Long before this book..." explanation
- Full quotes with attribution
- Deep philosophical content
- Complete manuscript excerpts
- "How to Read It" section

---

### Level 4: About Page (System Overview)
**Purpose:** Understand the broader system, not just the book  
**Content Strategy:**
- Different angle - about the S2S system itself
- Different content focus
- Console/Codex/System explanation
- Author information
- System architecture

**What to REMOVE from About page:**
- Book-specific content (that's on About-the-book)
- Manuscript quotes (those are on Books/About-the-book)
- Detailed book descriptions

**What to KEEP/ADD on About page:**
- System overview (different from book content)
- Core principles
- Four-layer system (Book/Codex/Console/Index)
- Author bio
- System vs. book distinction

---

### Level 5: Preorder Page (Practical Details)
**Purpose:** Transaction-focused, minimal philosophy  
**Content Strategy:**
- Focus on what you get, pricing, logistics
- Minimal philosophical content
- Practical information only
- FAQ focused on ordering

**What to REMOVE from Preorder page:**
- Philosophical quotes
- Long explanations
- Manuscript excerpts
- "Paradigm" content

**What to KEEP on Preorder page:**
- Brief book description (one sentence)
- What's included (practical list)
- Pricing
- Shipping info
- FAQ (practical questions)
- Order form

---

## Quote Distribution Strategy

### Homepage (1 unique quote)
- Use: Opening excerpt from manuscript (the "No one remembers the first moment..." one)
- Do NOT use: "A system reveals itself..." (move to About-the-book)

### Books Page (Multiple quotes, different from homepage)
- Use: Chapter excerpts
- Use: Interlude quotes
- Do NOT repeat: Homepage quote
- Do NOT repeat: "A system reveals itself..."

### About-the-Book Page (Full quotes with context)
- Use: "A system reveals itself..." (full context)
- Use: "Long before this book..." (full explanation)
- Use: Complete manuscript quotes with attribution
- Do NOT repeat: Homepage teaser quote
- Do NOT repeat: Books page chapter quotes

---

## Section Movement Plan

### Move FROM Homepage TO About-the-Book:
- Full "Paradigm" section (keep brief version on homepage)
- Detailed "Long before this book..." explanation

### Keep on Homepage (but make briefer):
- "What Book One Contains" → One sentence only
- "A Glimpse Into The Manuscript" → ONE unique excerpt
- "Who This Is For" → Keep (brief list)
- "About This Work" → Condensed version

### Remove from Books Page:
- "Paradigm" section (if present)
- Book description that matches homepage exactly

### Remove from About-the-Book Page:
- Brief teasers (those belong on homepage)
- Simple descriptions (those belong on books page)

---

## Implementation Steps

1. **Homepage Simplification**
   - Condense "Paradigm" section to brief version
   - Replace "A system reveals itself..." quote with unique manuscript excerpt
   - Make "What Book One Contains" one sentence
   - Keep "A Glimpse" but use different excerpt than other pages

2. **Books Page Enhancement**
   - Ensure all quotes are different from homepage
   - Add more manuscript excerpts
   - Remove any "Paradigm" content
   - Keep book description but vary wording slightly

3. **About-the-Book Page Deepening**
   - Add full "Paradigm" section (moved from homepage)
   - Add "A system reveals itself..." quote with full context
   - Ensure all quotes have full attribution
   - Add complete explanations

4. **About Page Refocusing**
   - Remove book-specific content
   - Focus on system architecture
   - Keep author bio
   - Emphasize Console/Codex/System

5. **Preorder Page Streamlining**
   - Remove philosophical content
   - Keep practical information only
   - Brief book description (one sentence)
   - Focus on ordering process

---

## Content Uniqueness Checklist

Before finalizing, ensure:
- [ ] No quote appears on more than one page
- [ ] "Paradigm" section only on About-the-book (brief version on homepage OK)
- [ ] Book description varies slightly on each page
- [ ] Homepage has unique manuscript excerpt
- [ ] Books page has different excerpts than homepage
- [ ] About-the-book has full quotes with context
- [ ] Preorder page has minimal philosophical content
